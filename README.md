# DeepLearningFinal
Proyecto final de Deep Learning. Resumen de curriculums con BART
Pasos de ejecucion:
  - Extracción de texto desde archivos PDF usando PyMuPDF
  - Preprocesamiento y construcción de dataset con pandas
  - Fine-Tuning de facebook/bart-large-cnn con PyTorch
  - Entrenamiento personalizado usando máscaras de atención y labels
  - Generación de nuevos resúmenes y evaluación del modelo con ROUGE

Tecnologías usadas:
Python, HuggingFace Transformers, PyTorch, Pandas, Google Colab
